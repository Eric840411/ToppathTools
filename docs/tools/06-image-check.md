# 圖片刪除驗證

**路由**：`/api/image-check/*`

---

## 功能說明

在 Playwright 內嵌瀏覽器中開啟遊戲頁面，自動驗證已刪除的圖片是否仍被載入。

---

## 操作流程

```
貼上已刪除圖片清單（blob URL 或路徑）
  └▶ 貼上前端 URL
       └▶ 啟動 Playwright 瀏覽器
            └▶ 截圖 / 點擊 / 捲動操作
                 └▶ 比對結果（哪些圖片仍存在）
```

---

## 操作項目

| 操作 | 說明 |
|------|------|
| 貼上已刪除圖片清單 | 輸入 blob URL 或路徑列表 |
| 貼上前端 URL | 遊戲頁面 URL，供 Playwright 開啟 |
| 啟動驗證 | 後台啟動 Playwright，自動截圖並比對 |
| 截圖 / 點擊 / 捲動 | 互動操作瀏覽器 |
